## Methods in Simplify

Import the following library in the Android Activity or class:

Import Statements:
 1) **For importing Primitives methods**
```Java
import com.arnold.simplify.Primitives;
```

2) **For importing All methods**
```Java
import com.arnold.simplify.*;
```
